# Python Game Launcher

Run the launcher directly with Python (no Thonny required):

```bash
python app.py
```

## What to do
- Keep built-in games in `games/` (currently only **Best Snake Game**).
- Add your own game `.py` files with **Developer Mode → Add Game Files**, or manually place files in `custom_games/`.
- Press **Refresh** in the launcher after adding games.
- In Developer Mode you can rename a selected game's launcher name.

## Developer PIN
- Default PIN is in `settings.json`.
- Change `developer_pin` there if needed.
